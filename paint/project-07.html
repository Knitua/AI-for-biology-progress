研究问题

抗冻肽数据非常稀缺，公开有效抗冻肽数量远少于抗菌肽、抗癌肽等体系。本项目的核心问题是：能否先从抗冻蛋白体系切入，批量发现 AFP-like 蛋白，再通过 motif 挖掘和序列下切得到潜在抗冻肽候选，并最终进入实验验证。

当前路线不是单纯预测已有抗冻肽，而是一个“预测-生成-筛选-motif-实验验证”的完整流程：先建立抗冻蛋白预测器，再训练生成器扩展候选空间，随后从天然蛋白库和生成蛋白库中筛选候选，最后用生物信息学工具挖掘功能 motif 并设计短肽。

研究意义

抗冻蛋白/肽能够调控冰晶生长和重结晶，在器官冷冻保存、细胞冻存、冷冻疗法和食品工业中有应用潜力。抗冻肽相比大蛋白更易合成、免疫原性更低，也更适合快速实验迭代。

该方向的亮点在于：抗冻肽体系目前缺少成熟的大规模 AI 筛选工作，如果能提出批量候选者并验证几条优质候选，就能形成清晰的方法学故事和应用价值。

项目目标

  • 构建多个 AFP 预测器，并通过多预测器级联降低假阳性。
  • 基于 ProtGPT2 + LoRA 生成 AFP-like 蛋白候选。
  • 从天然极地生物蛋白库和生成蛋白库中筛选高置信候选。
  • 使用 MMseqs2、k-mer、MAFFT、MEME 等工具挖掘保守 motif。
  • 将 motif 下切为候选抗冻肽，并推进冰重结晶抑制、结构表征和细胞冻存相关实验。

技术路线

  1. 预测器：使用 UniProt 约 7k-8k 正样本和约 8k 负样本训练，Swiss-Prot 作为独立验证集；模型包括 SaProt、ESM2、SaProt+ESM2 融合模型。
  2. 生成器：以 ProtGPT2 为基座，引入 LoRA/PEFT 微调，使模型更倾向生成 AFP-like 序列。
  3. 天然蛋白库筛选：从 UniProt 约 64 万条极地生物来源蛋白中，先用 ESM2-only 快速初筛，再用更强专家预测器二次判分。
  4. 生成蛋白库筛选：使用 LoRA 微调的 ProtGPT2 生成约 10000 条候选蛋白，再通过多个预测器筛选。
  5. motif 挖掘：对候选蛋白做 MMseqs2 去冗余/聚类、k-mer 统计、MAFFT 多序列比对、保守性分析和 MEME motif 挖掘。
  6. 实验验证：推进冰重结晶抑制实验、CD/Raman/DSC 结构与相互作用表征，以及细胞冻存与生物相容性实验。

当前进展

本项目已经形成比较完整的预测、生成、筛选和实验验证路线：

  • 已明确总体策略：由于抗冻肽直接数据稀缺，先挖掘抗冻蛋白，再从抗冻蛋白中识别功能 motif 并下切为候选抗冻肽。
  • 抗冻蛋白预测器开发已完成：目标是构建多个 AFP 预测器，对天然蛋白库和生成蛋白库进行轮询筛选，形成 predictor voting 框架。
  • 生成器已完成第一版：使用 ProtGPT2 + LoRA 微调后，生成序列经预测器评估的命中率约 815/1000；全量微调命中率约 497/1000；生成序列与训练数据不同源、不相似。
  • 天然蛋白库筛选已完成一轮：从 UniProt 约 64 万条极地生物来源蛋白中，用 ESM2-only 得到约 1800 条高置信候选，再计划结合更强预测器、去冗余和注释过滤。
  • 生成蛋白筛选已完成：LoRA 微调 ProtGPT2 生成约 10000 条蛋白候选，经多个预测器筛选得到约 8000 条 AFP-like 蛋白。
  • 统一生物信息学分析流程已完成：MMseqs2 去冗余与聚类、k-mer 统计、MAFFT 多序列比对、MEME motif 挖掘。
  • 已观察到一些 motif 共性：亲水/疏水氨基酸交替可能在冰面形成 M/W 式结构，重复 motif 片段可能倾向 alpha-helix。
  • 分子动力学模拟部分已有基本范式，但尚未完成。
  • 湿实验正在开展：包括冰重结晶抑制实验、CD/Raman/DSC 结构表征，以及细胞冻存和生物相容性实验。
