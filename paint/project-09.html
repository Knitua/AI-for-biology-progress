研究问题

本项目关注特殊亚型胃癌免疫治疗响应预测。核心问题不是简单输出“患者会不会响应”，而是给出临床医生能理解的决策路径：哪些空间结构、免疫微环境、TCR/BCR 克隆扩增、TLS、CD8 浸润、Treg 抑制等因素共同支持响应或不响应判断。

项目同时包含一个已经推进的 TLS 识别子任务：基于 WSI 切片和点标注，通过弱监督 patch 级训练生成 TLS 热图，为后续免疫微环境解释和响应预测提供关键概念特征。

研究意义

免疫治疗后区分响应者和非响应者是胃癌临床中的重要问题。这个项目的价值在于把多模态数据预测和机制解释结合起来，目标不是黑箱预测，而是形成可解释的患者报告。

如果做成，输出可以包括患者响应概率、H&E 全切片响应贡献热图、TLS/CD8/Treg/TCR 等概念瓶颈特征，以及关键区域标注。这比单纯多组学拼接预测更有临床沟通价值。

项目目标

  • 构建特殊亚型胃癌免疫治疗响应预测框架。
  • 整合 H&E 病理、基因组/表观组、scRNA、TCR/BCR、空间转录组和临床资料。
  • 参考 Multi-Embed，但加入 TCR/BCR 模态和概念瓶颈层，使模型中间输出具有生物学意义。
  • 完成 TLS 识别子任务，形成 WSI patch-level TLS 热图。
  • 输出结构化报告：TLS 高/低、CD8 浸润模式、TCR 克隆扩增、Treg/CD8 比例、关键区域标注等。

技术路线

  1. 多模态预测主线：以 Multi-Embed 为参考框架，针对 29 例小样本胃癌数据做迁移和适配。
  2. TCR/BCR 整合：简单方案是提取 Shannon 指数、Top10 克隆扩增度、CDR3 长度分布、V/J 基因偏好等 20-30 维标量；复杂方案是用 TCR-BERT/DeepTCR 编码序列 embedding，并与 scRNA/空间位置共同构图。
  3. 概念瓶颈层：在嵌入层和最终预测层之间加入 TLS 密度与成熟度、CD8 浸润模式、Treg/CD8 比例、新抗原负荷、TCR 多样性与扩增、MMR 蛋白表达模式等可解释概念。
  4. TLS 识别子任务：将 SVS/NDPI/MRXS 等 WSI 统一到 20x，切成 256×256 patch；对 TLS 点标注生成弱阳性 patch 标签。
  5. patch 表征：使用 CONCH/UNI/ViT 等病理基座提取 patch embedding，例如 ViT-small 为 384 维、CONCH 为 512 维。
  6. 热图输出：用已标注正负 patch 训练/对比，给待预测 WSI 的每个 patch 打分，输出 TLS 热图。

当前进展

已有进展包括：

  • 明确数据特点：当前约 29 例胃癌小样本，包含 H&E、基因组/表观组、scRNA、TCR/BCR、Xenium 空间转录组和临床资料。
  • 明确主要出口：特殊亚型胃癌免疫治疗响应预测 + 空间响应特征图谱。
  • 已设计临床可解释输出形式：患者响应概率、H&E 热图、个体化机制叙述、概念瓶颈画像和结构化报告。
  • 已提出相较 Multi-Embed 的两个关键改造：TCR/BCR 模态整合和特殊亚型概念瓶颈层。
  • TLS 识别已有初步实验：手头有 4 张已标注 WSI，采用 3 张训练、1 张预测的 leave-one-slide-out 思路，并已形成 AT/BT/ET/GT 的预测结果；另有 KT 切片预测尝试。
  • 已发现 TLS 标注问题：当前多为点标注而非完整区域 mask，因此采用点周围小半径弱阳性 patch 生成策略。
