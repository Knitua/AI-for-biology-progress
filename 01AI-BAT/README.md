## 研究问题

本项目围绕蝙蝠为什么能够同时具备飞行、长寿、病毒耐受和相对低炎症损伤等特殊表型展开。核心问题是从 Bat1K、比较基因组和免疫适应证据中，筛选出适合团队继续分析的候选基因、通路和物种组合。

第一阶段建议聚焦 **免疫与病毒耐受相关基因**：例如干扰素通路、ISG15、APOBEC、TLR、RIG-I/MDA5、STING/cGAS、NLRP 炎症小体、DNA 损伤修复等。这些方向与蝙蝠低炎症反应、病毒耐受和疾病抵抗密切相关，也更适合形成可执行的序列表征和候选基因筛选任务。

## 研究意义

蝙蝠是一个非常适合展示团队生物探索能力的系统：它有清晰的异常表型，有大规模基因组资源，也有明确的医学与免疫学价值。这个项目可以把“生物现象、公开组学资源、候选机制、计算分析路线”连接起来。

对老师汇报时，本项目的价值在于：我们已经不只是收集论文，而是完成了数据状态判断和第一版分析路线收敛，知道第一阶段不应从 TB 级原始 reads 开始，而应优先利用已组装、已注释的 genome/CDS/protein 数据开展小规模试点。

## 项目目标

- 建立 10-20 个代表性蝙蝠和非蝙蝠哺乳动物对照物种的小规模试点集。
- 从已有文献和数据库中整理免疫、病毒耐受、炎症调控和 DNA 损伤修复候选基因集。
- 基于已注释 protein/CDS 序列开展序列表征、聚类、分类或异常检测。
- 输出蝙蝠特异候选基因排序表、物种/基因 embedding 可视化和后续扩大到 Bat1K 更多物种的建议。

## 技术路线

1. 以 Bat1K 官网和 NCBI BioProject `PRJNA489245` 为主要数据入口，优先使用已组装和注释数据。
2. 选择测序、组装、注释均完成的物种；当前可用物种状态表覆盖 1469 个物种/条目，其中 91 个已完成测序、组装和注释。
3. 第一阶段选取 6-12 个蝙蝠物种和 3-8 个非蝙蝠哺乳动物对照，避免一开始处理 1.73 TB 级 SRA 原始数据。
4. 构建免疫与病毒耐受候选基因清单，并提取各物种 ortholog 的 protein/CDS 序列。
5. 使用 ESM2 进行蛋白序列表征，必要时加入 DNABERT-2 或 Nucleotide Transformer 做 CDS/调控区辅助分析。
6. 通过 PCA/UMAP、聚类、分类模型、异常检测和功能富集，筛选蝙蝠中特异偏离或值得进一步验证的候选基因。

## 当前进展

本项目已经形成比较完整的前期基础：

- 已建立 Bat1K 官网和 NCBI BioProject 数据入口。
- 已完成蝙蝠物种测序状态统计：1469 个条目、158 个测序完成、126 个组装完成、91 个注释完成且三项均完成。
- 已完成第一版深度学习分析方案，明确第一阶段不从原始 reads 起步，而是从已注释 protein/CDS 数据切入。
- 已初步确定推荐主题：基于预训练基因组/蛋白模型，对蝙蝠免疫和病毒耐受相关基因进行序列表征与候选基因筛选。

项目汇报时应将老药新用相关结果与 AI-BAT 主线区分，避免把药物筛选内容混入蝙蝠基因组分析。
